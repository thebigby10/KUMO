GET /api/labs - Get all labs (with optional archived query param)
[
/api/labs	only active labs
/api/labs?archived=false	only active labs
/api/labs?archived=true	only archived labs
/api/labs?archived=all	all labs
]

GET /api/labs/{id} - Get a specific lab by ID

POST /api/labs - Create a new lab

PUT /api/labs/{id} - Update entire lab

PATCH /api/labs/{id} - Partially update lab

PATCH /api/labs/{id}/archive - Archive lab

PATCH /api/labs/{id}/activate - Activate archived lab

DELETE /api/labs/{id} - Delete lab
